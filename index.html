📄 JSON 格式範例（含未來可擴充欄位）：

[
  {
    "date": "2026-04-20",
    "city": "台中",
    "type": "single_choice",
    "question": "「apple」的中文意思是什麼？",
    "question_pinyin": "「apple」 de Zhōngwén yìsi shì shénme?",
    "options": ["蘋果", "香蕉", "葡萄", "西瓜"],
    "options_pinyin": ["píngguǒ", "xiāngjiāo", "pútáo", "xīguā"],
    "answer": 0,
    "hint": "",
    "multi_answers": [],
    "short_answer": "",
    "tags": [],
    "difficulty": ""
  }
]

目前系統主要使用：date/question/options/answer； 若有提供 city 會顯示在題目卡片上； 其餘欄位可先留白，方便未來擴充提示、多選題、簡答題等功能。
